# Extension

Chrome extension popup for:

- generating WebRTC offer QR for mobile pairing
- accepting mobile answer payload
- receiving latest listing directly from mobile app
- filling marketplace forms via content adapters

## Run

```powershell
npm install
```

Load unpacked in `chrome://extensions`.
